# 📊 Sales Data Analysis using SQL

## 📌 Project Overview
This project analyzes sales data using SQL to extract meaningful business insights. It demonstrates skills in SQL querying, data analysis, and problem-solving.

## 🛠 Tools & Technologies
- MySQL
- SQL

## 📂 Dataset
The dataset includes:
- Order ID
- Product Name
- Category
- Sales Amount
- Quantity
- Order Date

## 🔍 Key Analysis
- Total sales calculation
- Top-selling products
- Category-wise sales
- Monthly sales trends
- Average sales

## 📈 Key Insights
- Electronics category generated the highest revenue
- Laptop is the top-selling product
- Sales show an increasing trend over months

## 📁 Project Files
- sales_data.csv
- queries.sql
- insights.txt
